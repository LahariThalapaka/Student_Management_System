# Student Management System

## Description
A console-based CRUD application developed using Java, JDBC, and MySQL.

## Features
- Insert Student
- Update Student
- Delete Student
- Display All Students
- Student ID validation (positive check)
- Menu-driven implementation using switch-case and do-while loop

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE
